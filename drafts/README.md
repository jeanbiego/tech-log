# drafts

このディレクトリは記事の下書きを保存するための場所です。ユーザーはここにラフなメモや本文草案を置きます。

- 下書きごとに `yyyymmdd_topic` の形式でサブディレクトリを作成してください。
  - 例: `20260302_getting-started`
- 下書きファイルは `draft.md` という名前で配置します。
- 画像や図表などの資産は `assets/` サブディレクトリに保存してください。
- 下書きは Zenn に公開されませんが、Codex が記事化する際の元ネタとして参照します。

プレビューを行いたい場合は、Codex が自動的に `articles/` ディレクトリに `_draft-<slug>.md` を生成し、Zenn CLI で確認できるようにします。
