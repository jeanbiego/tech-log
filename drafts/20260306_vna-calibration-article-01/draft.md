# VNAで何を測る？
## TL;DR

- VNAは、回路に電気信号を入れたときの入出力を測定する装置  
- 実際の測定にはケーブルやコネクタなどの影響が含まれる  
- その影響を数学的に取り除くのが校正


## 1. RFのための測定機たち

電気信号、その中でも高周波（RF）を測る装置には、様々なものがあります。
主な装置は以下の三種類ですね。
| 測定機 | 主な用途 |
| --- | --- |
| オシロスコープ | 時間波形 |
| スペクトラムアナライザ | 周波数スペクトル |
| VNA | 回路の伝送特性 |

この中でも、VNAは回路の入出力を測定する装置です。

たとえば、こんなときに使います。
- フィルタが通す周波数はどこからどこまで？
- アンプの利得はいくつ？
- コネクタと基板はマッチングしてる？

## 2. RF回路で知りたいこと

RF回路を評価するとき、実際に知りたいのは次のような特性です。

-   挿入損失 (Insertion Loss)
-   反射 (Return Loss)
-   クロストーク

これらは **Sパラメータ**として表現されます。

例

S21 → 挿入損\
S11 → 反射\
S31 → クロストーク


## 3. Sパラメータとは何か

Sパラメータは

### 入射波と反射波の比

として定義されます。

最低限の式

S21 = b2 / a1

ここでは詳細な波の定義には踏み込まず、
「入力信号に対してどれだけ出力へ伝わるか」という
直感的理解を重視します。

------------------------------------------------------------------------

## 4. VNAはどうやって測っているのか

VNAは基本的に次のことを行っています。

1.  Port1から信号を出す
2.  Port1で反射を測る
3.  Port2で透過を測る

この結果として

S11、S21 などが測定されます。

------------------------------------------------------------------------

## 5. 実際の測定系

実際の測定では、DUTだけでなく測定系全体が含まれます。

VNA → ケーブル → アダプタ → DUT → ケーブル → VNA

この測定系の影響が、測定結果に含まれます。

------------------------------------------------------------------------

## 6. エラーボックスモデル

測定系の影響は、数学的には

### Error Box

として表現されます。

Port → Error Box → DUT → Error Box → Port

このモデルにより、測定系の影響を数式で扱うことができます。

------------------------------------------------------------------------

## 7. 校正とは何か

校正とは

### 測定系の影響（Error Box）を求めて取り除く操作

です。

校正前

Error Box → DUT → Error Box

校正後

DUT

------------------------------------------------------------------------
