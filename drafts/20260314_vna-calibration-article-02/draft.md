# VNAのReference Planeとは何か

## TL;DR

- VNA測定では「どこからどこまでをDUTとするか」を決める必要がある
- その測定の起点を **Reference Plane（基準面）** と呼ぶ
- 校正とは **誤差を取り除くと同時に、基準面を決める作業** でもある

## 0. 前回の記事からのつながり

前回の記事では、VNA校正の基本について説明しました。

- VNA測定には多くの誤差が含まれる
- それらをError Boxという形でまとめて数学的に取り除く
- それがVNAの校正

今回はその続きとして、次の疑問に答えます。

> VNAは「どこからどこまで」を測っているのでしょうか？

この記事では、この測定の起点になる **Reference Plane（基準面）** について説明します。

## 1. 普通に測定するとどうなるか

まず、典型的なVNA測定系を見てみます。

![VNA測定系の概略図](images/vna_measurement_system.png)

測定系は大体このようになっています。

```text
VNA - cable - connector - DUT - connector - cable - VNA
```

このとき、VNAが測定しているのは **DUTだけではありません。**

測定結果には次のすべてが含まれます。

- ケーブル
- コネクタ
- 基板
- DUT

つまり、実際の測定は次の状態です。

```text
measurement = cable + connector + DUT + connector + cable
```

では、この測定から **DUTだけの特性** を取り出すにはどうすればいいのでしょうか。

そこで登場するのが **校正（calibration）** です。

## 2. 校正とは基準面を決めること

VNA校正には大きく2つの役割があります。

1. 測定系の誤差を取り除く
2. **測定の基準面（Reference Plane）を決める**

この基準面がどこに置かれるかによって、**VNAが見ているDUTの範囲が変わります。**

## 3. SOLT校正: 基準面はコネクタ端

同軸測定では、一般的に **SOLT校正** を行います。

![SOLT校正キットの概略図](images/solt_calibration_kit.png)

SOLTとは次の4つの標準です。

- Short
- Open
- Load
- Thru

これらをVNAケーブルの先端に接続して校正します。

つまり校正は次の位置で行われます。

```text
VNA - cable - connector
```

すると、校正後の測定は次のようになります。

```text
reference plane
      |
connector | DUT | connector
```

つまり、この測定でVNAが見ているDUTは次の範囲です。

```text
connector + DUT + connector
```

## 4. TRL校正: 基準面をDUT近くに動かす

しかし実際の回路評価では、次のように考えることがよくあります。

> コネクタやフィクスチャの影響を除いて、**DUTだけの特性** を知りたい

そこで使われるのが **TRL校正** です。

![TRL校正構造の概略図](images/trl_calibration_structures.png)

TRLでは次の3つの構造を用意します。

- Thru
- Reflect
- Line

これらを **DUTと同じ基板上** に作っておきます。

校正はこの基板上で行われます。すると、校正後の基準面は次のようになります。

```text
connector - trace | DUT | trace - connector
                    ^
              reference plane
```

つまり、基準面が **DUTのすぐ近く** に移動します。

これにより、次のような要素の影響を取り除いた測定が可能になります。

- コネクタ
- フィクスチャ
- 配線

## 5. 校正とは基準面を動かすこと

ここまでの話をまとめると次のようになります。

| 校正方法 | 基準面 |
| --- | --- |
| SOLT | コネクタ端 |
| TRL | DUT近傍 |

つまりVNA校正とは、次の2つの意味を持っています。

- 測定系の誤差を取り除く
- **測定の基準面を決める**

## 6. まとめ

この記事では、次の3つの関係を整理しました。

- VNA測定

校正は単に誤差を減らすだけではなく、**どこをDUTとして扱うかを定義する作業** でもあります。
